## Installation

[![cka](https://nodei.co/npm/cka.png)](https://npmjs.org/package/cka)

```
npm install cka -g
or
yarn global add cka
```

## Creating an App

```
npx cka new myApp // npm 5.2+ required
cka new myApp
cka new myApp --no-install
```
